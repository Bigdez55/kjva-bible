# Project Identity Decision Record

Use this file as a template when a consuming project needs to name or re-scope a
Tokenless model deployment.

## Record

| Field | Value |
|---|---|
| Project | `<project-name>` |
| Model export | `<model-id>` |
| Runtime owner | `<owner/operator>` |
| Deployment target | `<local/cloud/edge>` |
| Governance policy | `<policy reference>` |
| Memory policy | `<storage reference>` |
| Date | `<yyyy-mm-dd>` |

The decision belongs to the consuming project, not to this neutral model repo.
