## Scope

### Issue #1: Deprecated Non-Secure Unarchive Path
**File:** `Apollo16/GLOBALS/MANAGERS/DefaultsUtility.swift`, `getNotes()` function  
**Severity:** CRITICAL (Security + Correctness)

**Current Code:**
```swift
guard let decodedData = try NSKeyedUnarchiver.unarchiveTopLevelObjectWithData(codedData!) as? Array<Note> else {
    // ...
}
```

**Problems:**
1. `unarchiveTopLevelObjectWithData` is deprecated since iOS 12
2. Bypasses secure coding entirely (even though Note implements NSSecureCoding)
3. Force-unwraps `codedData` which is optional
4. No error handling for decoding failures

**Fix:**
```swift
guard let codedData = UserDefaults.standard.data(forKey: "notes") else {
    return []
}

do {
    let unarchiver = try NSKeyedUnarchiver(forReadingFrom: codedData)
    unarchiver.requiresSecureCoding = true
    
    guard let notes = try unarchiver.decodeTopLevelObject(
        of: [NSArray.self, Note.self],
        forKey: NSKeyedArchiveRootObjectKey
    ) as? [Note] else {
        print("Failed to decode notes: expected NSArray of Note objects")
        return []
    }
    
    return notes
} catch {
    print("Error decoding notes: \(error.localizedDescription)")
    return []
}
```

**Test:**
- Load app, verify notes appear
- Force-kill app mid-save, restart, verify notes still load (no crash)
- Manually corrupt a few bytes in the UserDefaults data, restart, confirm graceful fallback

### Issue #3: Add Error Recovery for Partially Corrupted Archives
**File:** `Apollo16/GLOBALS/MANAGERS/DefaultsUtility.swift`, `getNotes()` enhancement  
**Severity:** HIGH (Recoverability)

**Enhancement:**
If `getNotes()` detects a partially corrupted archive, skip the corrupted note and continue loading others instead of failing entirely.

**Implementation:**
```swift
func getNotes() -> [Note] {
    guard let codedData = UserDefaults.standard.data(forKey: "notes") else {
        return []
    }
    
    do {
        let unarchiver = try NSKeyedUnarchiver(forReadingFrom: codedData)
        unarchiver.requiresSecureCoding = true
        
        guard let notesArray = try unarchiver.decodeTopLevelObject(
            of: [NSArray.self, Note.self],
            forKey: NSKeyedArchiveRootObjectKey
        ) as? [Note] else {
            return []
        }
        
        // Filter out any nil entries (partial corruption)
        let validNotes = notesArray.compactMap { $0 }
        
        if validNotes.count < notesArray.count {
            print("Warning: \(notesArray.count - validNotes.count) corrupted notes skipped")
        }
        
        return validNotes
    } catch {
        print("Error decoding notes: \(error.localizedDescription)")
        // Fallback: attempt to salvage by creating backup
        backupCorruptedNotes()
        return []
    }
}

private func backupCorruptedNotes() {
    // Log the corrupted data for debugging
    if let data = UserDefaults.standard.data(forKey: "notes") {
        let timestamp = ISO8601DateFormatter().string(from: Date())
        UserDefaults.standard.set(data, forKey: "notes_corrupted_\(timestamp)")
        print("Corrupted notes backed up to: notes_corrupted_\(timestamp)")
    }
}
```

**Test:**
- Corrupt one note in the archive (hex-edit a few bytes in the middle)
- Restart app
- Verify: corrupted note is skipped, other notes load, warning appears in console, backup is created
