### Step 4: Code Migration for Alamofire 5.x

**Alamofire 4 → 5 API Changes:**

**Old (Alamofire 4):**
```swift
Alamofire.request(urlString).responseJSON { response in
    switch response.result {
    case .success(let value):
        // Handle value
    case .failure(let error):
        // Handle error
    }
}
```

**New (Alamofire 5):**
```swift
AF.request(urlString).responseJSON { response in
    switch response.result {
    case .success(let value):
        // Handle value
    case .failure(let error):
        // Handle error
    }
}
```

**Files to Update:**
- `Apollo16/NETWORK/AFWrapper.swift` - Update all `Alamofire.request(...)` to `AF.request(...)`
- Remove `import Alamofire`, replace with `import Alamofire as AF` if needed

**Example Changes:**
```swift
// Old
Alamofire.request(urlString).responseJSON { response in
    guard let json = response.result.value as? [String: Any] else {
        completion([])
        return
    }
}

// New
AF.request(urlString).responseJSON { response in
    guard let json = response.result.value as? [String: Any] else {
        completion([])
        return
    }
}
```

**Validation:**
```bash
grep -n "Alamofire\.request" Apollo16/NETWORK/*.swift
# Should return 0 results (all changed to AF.request)
```

### Step 5: Replace SwiftIcons with SF Symbols

**Old (SwiftIcons):**
```swift
let icon = UIImage.init(icon: .fontAwesomeBrands(.itunes), size: CGSize(width: 40, height: 40))
```

**New (SF Symbols):**
```swift
let icon = UIImage(systemName: "music.note")
```

**Mapping (Common Icons):**
- `.fontAwesomeBrands(.itunes)` → `"music.note"` or `"music.note.list"`
- `.fontAwesome(.trash)` → `"trash"`
- `.fontAwesome(.edit)` → `"pencil"`
- `.fontAwesome(.save)` → `"checkmark.circle"`
- `.fontAwesomeBrands(.dropbox)` → `"square.and.arrow.down"`

**Files to Update:**
- `Apollo16/NIBS/Bottom Sheet/ChooseSource.swift` (iTunes and Dropbox icons)
- Any other files using SwiftIcons

**Validation:**
```bash
grep -r "SwiftIcons\|UIImage.init(icon:" Apollo16/ --include="*.swift"
# Should return 0 results
```
