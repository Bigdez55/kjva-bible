### Issue #2: Dropbox listFolder Called Twice — Root Files Duplicated
**File:** `Apollo16/GLOBALS/Functions.swift`, `fetchFiles()` function  
**Severity:** MEDIUM (Data Quality)

**Current Code:**
```swift
func fetchFiles() {
    // Call 1
    client.files.listFolder(path: "", recursive: false) { response in
        array_dropboxFiles.append(contentsOf: response.entries)
        
        // Call 2 (nested)
        client.files.listFolder(path: "", recursive: true) { response2 in
            array_dropboxFiles.append(contentsOf: response2.entries)
        }
    }
}
```

**Problem:** Both recursive and non-recursive calls append to same array. Root files appear twice.

**Fix:**
```swift
func fetchFiles() {
    // Single call with recursive: true
    client.files.listFolder(path: "", recursive: true) { response in
        array_dropboxFiles = response.entries  // Replace, don't append twice
    }
}
```

### Issue #3: ChooseSource Bottom Sheet Animation Position Math
**File:** `Apollo16/NIBS/Bottom Sheet/ChooseSource.swift`, `show()` function  
**Severity:** MEDIUM (Layout Bug)

**Current Code:**
```swift
func show() {
    self.center.y -= self.bounds.height
}

func hide() {
    self.center.y += self.bounds.height
}
```

**Problem:** Moves center by height, which may overshoot or undershoot depending on frame timing. Hardcoded values in rotation handler don't match actual position.

**Fix:**
```swift
func show() {
    guard let parentView = parentView else { return }
    
    let targetY = parentView.bounds.height - (self.bounds.height / 2)
    
    UIView.animate(withDuration: 0.3) {
        self.center.y = targetY
        self.layoutIfNeeded()
    }
}

func hide() {
    guard let parentView = parentView else { return }
    
    let targetY = parentView.bounds.height + (self.bounds.height / 2)
    
    UIView.animate(withDuration: 0.3) {
        self.center.y = targetY
        self.layoutIfNeeded()
    }
}
```

### Issue #6: Replace IUO Delegates with Optional Chaining
**Files:** Multiple delegate properties throughout  
**Severity:** MEDIUM (Crash Prevention)

**Current Pattern:**
```swift
var delegate: MultiselectSnackDelegate!  // IUO
var delegate: AudioBoxDelegate!           // IUO
var delegate: TVC_SearchCellDelegate!     // IUO

// Usage:
delegate.deleteSelectedNotes()  // Crashes if delegate not set
```

**Problem:** If delegate not assigned before action, crashes with unhelpful "unexpectedly found nil" message.

**Fix:**
```swift
weak var delegate: MultiselectSnackDelegate?  // Weak optional
weak var delegate: AudioBoxDelegate?          // Weak optional
weak var delegate: TVC_SearchCellDelegate?    // Weak optional

// Usage:
delegate?.deleteSelectedNotes()  // Silently ignored if delegate is nil
```

**Files to Update:**
- `Apollo16/NIBS/Snackbar/MultiselectSnack.swift`
- `Apollo16/NIBS/Added Audio/AudioBox.swift`
- `Apollo16/NIBS/Search Result/TVC_SearchCell.swift`
- `Apollo16/NIBS/Bottom Sheet/ChooseSource.swift`
- Any other view with delegate pattern

### Issue #9: Magic Number View Tag
**File:** `Apollo16/NIBS/Bottom Sheet/ChooseSource.swift`  
**Severity:** LOW (Code Quality)

**Current Code:**
```swift
self.tag = 369
if let _ = self.parentView.viewWithTag(369) { return }
```

**Problem:** Magic number; if another view is assigned tag 369, guard fires incorrectly.

**Fix:**
```swift
private let chooseSourceTag = 369

init() {
    super.init(frame: .zero)
    self.tag = chooseSourceTag
}

func show() {
    if parentView.viewWithTag(chooseSourceTag) != nil {
        return  // Already showing
    }
}
```
