- DRIFT-STRUCT-03b: Backend `LoginHistoryResponse` uses `created_at/device_name`; frontend `LoginAttempt` expects `timestamp/device_fingerprint`.
- DRIFT-STRUCT-03c: Disable2FA frontend sends `{verification_code, method}`; backend expects `{code, password}`. HTTP 422.
