## Security Configuration
- **Password hashing**: security.py uses CryptContext with schemes=["argon2", "bcrypt"]
- argon2 is primary (OWASP recommended), bcrypt kept for legacy compatibility
- Standard bcrypt version: `>=4.0.0,<4.3.0` (passlib compatibility)
- Standard argon2-cffi version: `>=23.1.0`

## Test Database Isolation
- SQLite test.db used for testing (conftest.py)
- db_session fixture: scope="function" with create_all/drop_all per test
- Cleanup fixture removes test.db after session
- pytest.ini: asyncio_mode=auto, coverage threshold 80%
