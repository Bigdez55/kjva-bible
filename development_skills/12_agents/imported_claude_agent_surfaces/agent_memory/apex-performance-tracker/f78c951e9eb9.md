## Critical Data Facts

- TradeDecisionLog: 3,559 rows; 448 AI; 160 outcomes (4.5% fill rate)
- episode_terminal: ALL 160 labeled rows = True — corrupted, do not use for DRL
- signal_source: 'ai' | 'rule_based' (authoritative field, not confidence-inferred)
- strategy_name: exists in model, populated by auto-trading loop
- win_rate from backend: 0.0–1.0 FRACTION (frontend expects 0–100 PERCENT)

## P0 Schema Mismatch

Backend `GET /analytics/performance-summary` returns: `{daily_pnl[], win_rate(fraction), snapshot_count}`
Frontend `PerformanceSummary` expects: `{sharpe_ratio, sortino_ratio, calmar_ratio, information_ratio, win_rate(%), profit_factor, expectancy, max_drawdown_pct, monthly_returns[]}`
No Sharpe/Sortino/Calmar computed anywhere in backend.

## Equity Curve is Mock Data

PerformanceDashboard renders MOCK_EQUITY_DATA (Math.random) unconditionally — never uses RTK Query data. BenchmarkChart defaults to MOCK_DATA hardcoded at 2026-03-02.

## Risk-Free Rate

Current: 4.0% (wrong). Required: 4.5% current T-bill. Formula: `0.045 / 252`. Annualization: 252 trading days (correct).

## $500/Day Target

At wr=53.5%, avg_win=$75, avg_loss=$50, trades=Poisson(7): P10=-$93, P50=$118, P90=$329. Target unreachable at P50. Need 65% win rate + 20 trades/day.
