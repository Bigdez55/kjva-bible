## Type Mismatches
- V2 SubscriptionPlan: {id, name, price, apy, trades, features, popular}
- API SubscriptionPlan: {id, name, price_monthly, price_yearly, features, popular}
- Fields `price`, `apy`, `trades` do not exist on API response

## API Call Pattern on Dashboard Mount
1. GET /api/v1/trading/batch-data (30s cache)
2. GET /api/v1/market-enhanced/quotes?symbols=AMZN,META,AMD,NFLX,DIS (60s cache)
- Only 2 concurrent requests -- well-optimized

## Alpha Vantage Rate Limits
- Free: 25/day, 5/min
- Backend /market-enhanced/quotes batches symbols into single call
- Need server-side Redis caching + dedup for production scale
- Recommend upgrading to paid tier for >10 concurrent users
