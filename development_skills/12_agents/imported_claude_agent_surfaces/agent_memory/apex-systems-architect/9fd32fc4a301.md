- Must use UEFI path (OVMF) — not `-kernel` flag (boot protocol requires EFI memory map)
- BSS-01 MUST be fixed first (static globals will be random without BSS clear)
- IDT-01 acceptable in QEMU (stacks rarely overflow to zero)
- SWAPGS-01 acceptable in QEMU (no NMIs by default)
- Serial output ready (COM1 115200 8N1 via serial_early_puts)
