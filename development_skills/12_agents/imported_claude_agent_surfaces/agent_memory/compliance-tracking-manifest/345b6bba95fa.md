## CRITICAL FINDINGS (Must Fix Before Any Sprint Work)

### CRITICAL-1: Circuit Breaker Gate Never Fires -- FIXED (2026-03-08)
- **File:** `backend/app/services/auto_trading_service.py:333` + `trade_executor.py` (3 locations)
- **Bug:** `circuit_breaker.check()` returns `Tuple[bool, CircuitBreakerStatus]` -- always truthy
- **Impact:** Trading continued through ALL adverse conditions -- gate was NEVER firing in entire trading history
- **Resolution:** Tuple unpacked in all 4 locations. `auto_trading_service.py:550` was fixed in prior sprint; `trade_executor.py` (3 locations) fixed in this sprint.

### CRITICAL-2: Circuit Breaker State Lost on Container Restart
- **File:** `backend/app/trading_engine/engine/circuit_breaker.py:160,174-190`
- **Bug:** JSON file persistence on ephemeral Cloud Run filesystem
- **Impact:** Tripped breakers silently reset on every restart/scale/deploy
- **Agents:** Fintech Auditor (CRITICAL-2), Security Sentinel (S5), Architect (File 1, Issue 1)
- **Fix:** Migrate `_load_status`/`_save_status` to PostgreSQL table

### CRITICAL-3: threading.RLock Blocks Event Loop
- **File:** `backend/app/trading_engine/engine/circuit_breaker.py:74`
- **Bug:** `threading.RLock()` called from 11 async call sites across 5 files
- **Impact:** Every lock acquisition blocks the entire asyncio event loop
- **Agents:** Quant Architect (20 call sites mapped), Architect (File 1, Issue 2)
- **Fix:** Dual-mode: `asyncio.Lock` wrapper for async callers, keep `threading.RLock` for 9 sync callers

### CRITICAL-4: No Shutdown Hook for Active Trading
- **File:** `backend/app/main.py` (lifespan shutdown block)
- **Bug:** SIGTERM kills active trading loops without cleanup
- **Impact:** Orphaned DB connections, uncommitted transactions, no position state save
- **Agents:** Architect (File 3, Issue 1), Security Sentinel (S5)
- **Fix:** Add `AutoTradingService` shutdown to lifespan teardown

### CRITICAL-5: Hardcoded Equity Values Block Crypto
- **Files:** `backend/app/services/broker/alpaca.py` (lines 232, 315, 506, 899, 943)
- **Bug:** `time_in_force: "day"`, `asset_type: "equity"`, `/stocks/` quote path, `us_equity` assets
- **Impact:** ALL crypto orders rejected, positions mislabeled, quotes 404
- **Agents:** Quant Architect (5 P0 blockers), Architect (File 4, Issues 1-6), Guardian Sniper (P0)
- **Fix:** Asset-class-aware routing in `_prepare_order_data`, `get_positions`, `get_quote`, `reconcile`, `list_tradable_assets`
