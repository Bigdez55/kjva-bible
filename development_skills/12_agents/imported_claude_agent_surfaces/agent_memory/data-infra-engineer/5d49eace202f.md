## PostgreSQL Trigger Template (for bot_audit_log)

```python
# In add_bot_safety_tables migration upgrade():
op.execute(sa.text("""
    CREATE OR REPLACE FUNCTION prevent_audit_log_modification()
    RETURNS TRIGGER AS $$
    BEGIN
        RAISE EXCEPTION
            'bot_audit_log is immutable. (row id=%)', OLD.id
            USING ERRCODE = 'restrict_violation';
        RETURN NULL;
    END;
    $$ LANGUAGE plpgsql;
"""))
op.execute(sa.text("""
    CREATE TRIGGER trg_bot_audit_log_no_delete
    BEFORE DELETE ON bot_audit_log
    FOR EACH ROW EXECUTE FUNCTION prevent_audit_log_modification();
"""))
op.execute(sa.text("""
    CREATE TRIGGER trg_bot_audit_log_no_update
    BEFORE UPDATE ON bot_audit_log
    FOR EACH ROW EXECUTE FUNCTION prevent_audit_log_modification();
"""))

# In downgrade():
op.execute(sa.text("DROP TRIGGER IF EXISTS trg_bot_audit_log_no_delete ON bot_audit_log;"))
op.execute(sa.text("DROP TRIGGER IF EXISTS trg_bot_audit_log_no_update ON bot_audit_log;"))
op.execute(sa.text("DROP FUNCTION IF EXISTS prevent_audit_log_modification();"))
```
