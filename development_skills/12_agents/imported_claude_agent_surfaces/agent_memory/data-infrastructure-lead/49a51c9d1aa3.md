# Sprint 4 Data Architecture Readiness Assessment (2026-03-05)

## XSTORE Key-Value Schema for XPKG
- xpkg_package_t is ~79KB static struct (16 deps * ~200B + 256 files * ~300B + metadata)
- Key: package name (char[64]) + version triple (3 x uint32_t) = composite key
- Content hash: SHA-256 (32 bytes), signature: Ed25519 (64 bytes)
- XPKG_MAX_INSTALLED=512, XPKG_MAX_GRAPH_NODES=512
- No timestamps on xpkg_package_t (PG-3 from sprint3 audit still open)

## SHA-256 Availability for XBLOB
- xsec_sha256() one-shot API confirmed in xsec.h line 65
- Streaming API: init/update/final confirmed
- FIPS 180-4 compliant with selftest vectors
- Also has SHA-512 for Ed25519

## PAL Thread Status
- pal_thread_create() is FULLY WIRED (line 516-536 of pal_aether.c)

- NOT a stub -- returns kernel thread handle, allocates PAL handle
- pal_thread_join, pal_thread_yield, pal_thread_sleep_ns all wired
