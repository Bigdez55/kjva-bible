## Project Architecture Snapshot

### Token System

- Base tokens: `/ui/xframe/runtime/design_tokens.h` (101 XTOKEN_* macros)
- Extension tokens: `/ui/xframe/widgets/design_tokens_net.h` (Sprint 3, additive)
- ARGB8888 (0xAARRGGBB) color format throughout
- Token naming: XTOKEN_COLOR_*, XTOKEN_SPACE_*, XTOKEN_RADIUS_*, XTOKEN_DURATION_*, XTOKEN_Z_*
- Pure preprocessor constants — no pal.h include needed
- Dark-only palette — NO light-mode architecture (P3 DSF-THEME-01)
- Token completeness: 87% (missing shadow, font weight, opacity scale, shell dims)
- Token adoption (weighted): ~55% (Color ~91% / Spacing ~38% / Radius ~89% / Duration 0%)

### JS Token Color Divergence (P2 open — DSF-TOK-10)

ts_bridge.c ts_inject_design_tokens() uses hardcoded hex strings that do NOT match design_tokens.h:
- colorPrimary="#7C3AED" (purple) vs XTOKEN_COLOR_ACCENT=0xFF3A8FE8 (blue)
- colorError="#DC2626" vs XTOKEN_COLOR_ERROR=0xFFE84040
- fontSizeSm=12 vs XTOKEN_FONT_SIZE_SM=14 (all font sizes offset by one scale step)
Fix: generate JS token values from design_tokens.h at build time.

### Animation Engine (display/xcomp/animation.c)

- 9 easing functions: linear, in/out quad, in/out cubic, decelerate (OUT_CUBIC), accelerate (IN_CUBIC), in-out-cubic
- Spring physics: semi-implicit Euler, settle detection
- Pool: 256 slots, spinlock-protected, framerate-independent via pal_time_now_ns()
- XANIM_DURATION_FAST/NORMAL/SLOW (120/220/340ms) match XTOKEN_DURATION_* values
  but animation.h defines own constants rather than consuming XTOKEN_* (DSF-TOK-01)
- CRITICAL: Zero xanim_start() / xanim_spring() calls anywhere in widget, shell, or app code
  — animation engine is 100% unused by UI layer (DSF-ANIM-01)

### Font Rendering (display/xcomp/fonts.c)

- TrueType/OpenType sfnt parser, cmap Format 4 (BMP only, U+0000-U+FFFF)
- Quadratic bezier outline, 4x vertical supersampling scanline rasterizer
- 256-slot LRU glyph cache, spinlock-protected
- xfont_load() is a stub (no VFS) — only xfont_load_mem() works (DSF-FONT-01)
- No compound glyphs, no bold/italic, no emoji (U+10000+), no kerning, no BiDi

### Vector Engine (display/xcomp/vector.c)

- Path: move_to, line_to, quad_to, cubic_to, close; pre-built: rect, rounded_rect, circle, ellipse
- Fill: non-zero winding, 4x AA; Stroke: butt/round/square caps, miter/round/bevel joins
- Path pool: 16 paths (small), 4096 commands, 8192 coords per path
- No gradient, no SVG parser, no clip path, no dash pattern, no transform matrix

### Accessibility Patterns

- WCAG focus ring tokens exist: XTOKEN_COLOR_FOCUS_RING, XTOKEN_BORDER_WIDTH_FOCUS
- XTOKEN_COLOR_TEXT_DISABLED fails WCAG 4.5:1 contrast (P1 DSF-A11Y-02)
- No aria_label field on xframe_node_t — screen reader support is zero (DSF-A11Y-01)
- dialog.c has keyboard focus trap; button.c has Enter/Space; dropdown.c has keyboard nav
- scroll.c has NO keyboard scroll handler
