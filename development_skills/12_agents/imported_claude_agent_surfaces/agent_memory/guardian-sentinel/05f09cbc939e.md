## Security Patterns (Canonical)

### Constant-Time Comparison
- Python: `hmac.compare_digest(a, b)` -- never `==` for secrets
- TypeScript/Node: `crypto.timingSafeEqual(Buffer.from(a), Buffer.from(b))`
- Kernel C: `crypto_memneq()` -- never `memcmp` for MACs
- Applied everywhere tokens/MACs are compared
- ALL PLATFORM SERVICES VERIFIED CLEAR (post-045f4e6 fixes)

### Auth Pattern
- FastAPI: `HTTPBearer` + `Depends(_require_auth)` on protected routes
- Raw HTTP: `_check_auth()` checking `Authorization: Bearer <token>`
- All use `GENOS_SERVICE_TOKEN` from env (k8s Secret: genos-secrets)
- Health endpoints (/health, /healthz) always unauthenticated for k8s probes

### CSS Selector Validation
- Allowlist regex: `/^[a-zA-Z0-9\s\-_.:#,\[\]='"~|+*>^()]+$/`, max 500 chars

### Orange Suite Preload Security
- AI apps: GENOS_SERVICE_TOKEN fail-hard at startup
- PII sanitization + HTML stripping before ALL AI calls
- ALWAYS explicit field extraction; never `...payload` or `JSON.stringify(payload)` on renderer input
- Token isolation: GENOS_SESSION_TOKEN in preload only

### RCU Lifecycle (Kernel)
- ALWAYS synchronize_rcu() or kfree_rcu() between pointer NULL and kfree
- Use put_unaligned_le32/get_unaligned_le32 for unaligned access
- list_del() MUST be under same lock as list_add (node->mr_lock for vdram MR list)

### URL Validation
- Environment-sourced URLs MUST validate scheme (http/https only)
- urllib.request supports file://, gopher://, ftp:// -- block all non-http(s)
- provenance_client.py: FIXED -- _ALLOWED_SCHEMES frozenset + urlparse check in __init__

### Security Efficiency Patterns
- JWT decode: cache with LRU(maxsize=256) keyed by (token, epoch//30) -- safe because JWT is immutable
- PEM normalization: MUST happen at module init, NOT per-request (sync/app.py violates this)
- bcrypt rounds: MUST be explicit via env var (IDENTITY_BCRYPT_ROUNDS), never rely on library default
- Rate limiting: per-username lockout is insufficient -- need per-IP via slowapi on auth endpoints
- Body size: uvicorn --limit-max-body-size or middleware -- Pydantic validation happens AFTER full body read
- K8s probes: /health for liveness (no DB), /healthz for readiness (DB check) -- prevents cascading restarts
- Schema/registry files: cache at startup when readOnlyRootFilesystem guarantees immutability

### Shared JWT Auth (genos_auth.py -- new @2110bc5)
- make_decode_bearer() factory: reads PEM at import time, returns FastAPI Depends target
- VERIFIED: algorithms=["EdDSA"], audience+issuer validated, typ=="access" enforced
- FIXED @01df259: empty public key env var now raises RuntimeError at factory creation time (P1-EMPTY-KEY-01 RESOLVED)
- Per-IP rate limiting: IMPLEMENTED on /auth/login (10 failures/15min window)
- ISSUE: /auth/token/refresh lacks per-IP rate limiting (P2-REFRESH-IP-01)
- SSE /chat/stream: auth+rate_limit+circuit_breaker all fire BEFORE generator starts -- correct pattern

### XISC JIT Security
- xisc_program_t has checksum[32] SHA-256 field -- NEVER VERIFIED by JIT
- Capability opcodes (0x30-0x32) are currently no-ops but will need verification when implemented
