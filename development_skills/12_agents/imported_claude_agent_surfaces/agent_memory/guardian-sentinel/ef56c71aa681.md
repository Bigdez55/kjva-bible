## P1 Blockers

### P1-XKABI-01: Handle layout overlap (CRITICAL)
- Lines 90-108, 132-139 in xkabi_capabilities.c
- HMAC (bytes [8:15]) overwrites rights bits 16-31 and generation field
- xkabi_verify_handle reads corrupted rights at line 138
- Affects all hardware rights >= bit 16 (STORAGE_WRITE, NET_SEND/RECV, GPU_EXEC, MESH_JOIN, ATTEST, DERIVE)
- Fix: Expand handle to 24 bytes or restructure to avoid overlap

### P1-MR-01: Missing synchronize_rcu() in vdram_mr_free()
- Lines 119-138 in mr_registry.c
- After RCU_INIT_POINTER(NULL), kfree(mr) without waiting for RCU grace period
- Use-after-free from concurrent vdram_mr_lookup_rkey() readers
- Fix: Add synchronize_rcu() or use kfree_rcu()

### P1-PROV-01: No URL validation on PROVENANCE_SERVICE_URL
- Line 36 in provenance_client.py
- urllib.request supports file://, gopher://, ftp:// -- SSRF risk
- Fix: Validate scheme is http/https only
