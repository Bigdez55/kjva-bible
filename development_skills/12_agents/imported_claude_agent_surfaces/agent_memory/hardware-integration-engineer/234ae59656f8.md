## HP EliteBook x360 1030 G4 — Advanced Hardware Features (2026-03-20)

### S10: Keyboard Backlight LED Control
- **EC Register 0xB0**: Brightness level (0x00=off, 0x01=dim, 0x02=medium, 0x03=bright)
- **EC Register 0xB1**: Auto-off timeout in seconds (0x00=always on, default 0x1E=30s)
- **API**: hp_kbd_backlight_set/get(), hp_kbd_backlight_set/get_timeout(), hp_kbd_backlight_pct_to_level()
- **UI**: SettingsDisplay.tsx keyboard backlight slider (0-100% mapped to 4 levels) + timeout select

### S11: Fingerprint Sensor (Stub)
- **Hardware**: Synaptics WBDI, USB VID:PID 06CB:00BD
- **Status**: Stub driver -- USB HID bulk transfer not yet implemented (Sprint 40+)
- **Limits**: Max 10 enrolled fingerprints, 256-byte template (SHA-256 of minutiae), min 3 minutiae anti-spoofing
- **API**: hp_fingerprint_status/enroll_begin/enroll_complete/verify/delete()
- **UI**: SettingsAccounts.tsx enrollment wizard with per-finger selection
- **Security**: Templates never leave device, encrypted at rest, constant-time comparison

### S12: Smart Card Reader (Stub)
- **Hardware**: Alcor Micro, USB VID:PID 058F:9540, CCID class 0x0B
- **Status**: Stub driver -- USB CCID bulk transfer not yet implemented (Sprint 40+)
- **Protocol**: ISO 7816 T=0/T=1, max ATR 33 bytes, max APDU 261 bytes
- **API**: hp_smartcard_status/power_on/send_apdu/power_off()
- **UI**: SettingsPrivacy.tsx smart card auth section with certificate display
- **Use cases**: PIV (FIPS 201), X.509 cert auth, digital signatures, SSH key storage

### Greeter/Lockscreen Integration
- **greeter.c**: 3 auth methods (XSHELL_AUTH_PASSWORD=0, FINGERPRINT=1, SMARTCARD=2)
  - set_auth_method() toggles UI between password fields and sensor prompts
  - on_fingerprint_result() / on_smartcard_result() fire login callback on success
- **lockscreen.c**: on_fingerprint() / on_smartcard() bypass PIN entry
  - Biometric/smartcard failures do NOT increment PIN lockout counter
  - Fallback to PIN always available
