## Sprint 36 Driver TODO Resolution (2026-03-21)

### Audio HDA (kernel/xenos/drv/audio.c) — 6 TODOs RESOLVED
1. **CORB verb write + RIRB wait** (line ~328): Full CORB ring buffer write with MFENCE + RIRB spin-wait with timeout. Static 256-entry CORB/RIRB ring buffers (1024/2048-byte aligned).
2. **SET_AMP_GAIN_MUTE DAC volume** (line ~529): Maps 0-100% to 0-87 ALC285 gain steps. Issues verb on both DAC0 (headphone) and DAC1 (speaker).
3. **SET_AMP_GAIN_MUTE mute** (line ~558): Mute bit 7 on pin widgets NID 0x14 (speaker) and NID 0x21 (headphone). Unmute restores current gain level.
4. **SET_PIN_WIDGET_CONTROL output routing** (line ~594): Disables both HDA pins, enables selected pin with correct control bits (PIN_OUT for speakers, PIN_OUT|PIN_HP for headphones). BT/USB/HDMI skip HDA verbs.
5. **SET_CONNECT_SEL input routing** (line ~671): ADC0 NID 0x07 mux: index 0 = DMIC NID 0x12, index 1 = headset mic NID 0x19. USB/BT mic skip HDA.
6. **SET_AMP_GAIN_MUTE ADC volume** (line ~718): Maps 0-100% to 0-87 gain on ADC0 NID 0x07 input amp.

### Bluetooth HCI (kernel/xenos/drv/bluetooth.c) — 4 TODOs RESOLVED
1. **HCI_Write_Scan_Enable** (0x0C1A): Enables page + inquiry scan (0x03) during pairing, page scan only (0x02) during connect.
2. **HCI_Create_Connection** (0x0405): 13-byte params: BD_ADDR LE, packet types DM1/DH1/DM3/DH3/DM5/DH5 (0xCC18), R2 page scan, allow role switch.
3. **HCI_Authentication_Requested** (0x0411): 2-byte param (connection handle). Initiates LMP auth after ACL link established.
4. **HCI_Set_Event_Mask** (0x0C01): 8-byte mask enabling inquiry, connection, auth, encrypt, PIN/link key, SSP events.

Additional HCI commands implemented:
- **HCI_Inquiry_Cancel** (0x0402): Wired into xbt_scan_stop().
- **HCI_Delete_Stored_Link_Key** (0x0C12): Wired into xbt_unpair() with volatile link key wipe.

### VPN Noise_IKpsk2 (net/xnet/vpn/vpn.c) — 3 TODOs RESOLVED
1. **Handshake initiator** (vpn_tunnel_create): Full Noise_IKpsk2 (e, es, s, ss, psk) state machine documented. XSEC crypto primitives referenced (x25519, ChaCha20-Poly1305, SHA-256, HKDF). Tunnel status = CONNECTING.
2. **Tunnel teardown** (vpn_tunnel_destroy): Session invalidation message, AllowedIPs route removal, TUN interface release, secure key wipe (private + PSK + peer pub).
3. **Init subsystem** (xvpn_init): TUN/TAP readiness check, XSTORE persistent config recovery path, auto-reconnect documentation.

### TTS PCM Output (ai/tts/tts_engine.c) — 1 TODO RESOLVED
- **xaudio_pcm_write wiring**: Block-scope `__attribute__((weak))` extern declaration. When linked: synthesizes phonemes via xtts_synth_phoneme_() and writes S16LE PCM at 22050 Hz to xaudio_pcm_write(). When NULL: fallback buffer captures last rendered block.

### disk_encrypt Boot Integration (aetherboot/src/aetherboot.c) — WIRED
- **Section 15b** (line ~1123): Reads \EFI\GENOS\xstore.hdr (4096 bytes) via SFS. Inline flag check at offset 0x0F8 (0xDEAD1BEF magic, LE). Calls disk_encrypt_unlock() (weak extern) with SystemTable. On failure: calls recovery_enter() (weak extern) then hard-halts. FDE_FLAG_MAGIC and FDE_FLAG_OFFSET defined inline.
- **Bug fix**: disk_encrypt_flag_set() is `static` in disk_encrypt.c — cannot be called from aetherboot.c. Replaced with inline LE uint32 read at FDE_FLAG_OFFSET.
