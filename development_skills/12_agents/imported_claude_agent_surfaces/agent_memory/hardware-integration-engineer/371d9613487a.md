## HP EliteBook x360 1030 G4 -- Driver Gap Closure (2026-03-21)

### Gaps CLOSED in this session:

#### P1-01 CLOSED: Tiger Lake -> Whiskey Lake comment fix
- elitebook_x360.c: header comment corrected from "Tiger Lake / Alder Lake" to "Whiskey Lake"
- All EC register comments updated to reference Whiskey Lake platform
- Thunderbolt section updated from "Tiger Lake TBT" to "Whiskey Lake TBT"
- audio.h: updated from "Intel Tiger Lake" to "Intel Whiskey Lake / Cannon Point-LP"
- audio.c: updated PCI DID from 0x02C8 to 0x9DC8 (Cannon Point-LP HDA)

#### NEW: elitebook_x360.h header created
- Full public API declaration for all 15 sections of elitebook_x360.c
- Includes forward declarations, enums, and suspend/resume API
- Compile-verified: PASS

#### NEW: fingerprint.h + fingerprint.c created
- Hardware-independent XFPRINT API wrapping the low-level EC/USB layer
- API: init, status, enroll, enroll_with_hash, verify, verify_hash, delete, list_fingers
- Constant-time template comparison (anti-timing-attack)
- 10 finger positions with human-readable enum (right thumb..left little)
- Compile-verified: PASS

#### NEW: nfc.h + nfc.c created
- NFC controller driver for optional Broadcom BCM20795
- NCI (NFC Controller Interface) protocol implementation
- I2C transport stub (Intel LPSS I2C, PCI 8086:9DE8/9DE9)
- API: init, poll_start/stop, get_tag_info, transceive, read_ndef, deactivate
- Graceful degradation to NOT_PRESENT (many SKUs lack NFC)
- Compile-verified: PASS

#### NEW: smartcard.h + smartcard.c created
- Higher-level XSCARD API wrapping the USB CCID layer
- ISO 7816-4 APDU construction helpers (Case 1/3/4)
- PIV application support (FIPS 201): SELECT, VERIFY PIN, READ CERT
- PIV AID: A0 00 00 03 08 00 00 10 00
- Certificate containers: 0x9A (auth), 0x9C (sign), 0x9D (key mgmt), 0x9E (card auth)
- PIN secure zero after verify
- Compile-verified: PASS

#### NEW: wifi.h + wifi.c created
- Intel Wireless-AC 9560 driver stub
- Correct PCI IDs: VID=8086 DID=A370 (CNVi, Whiskey Lake PCH) and DID=2526 (discrete)
- CSR register map documented (HW_IF_CONFIG, INT, RESET, GP_CNTRL, HW_REV, FW_ERROR)
- Firmware loading path documented (TLV-based .ucode format, DMA to SRAM)
- TX/RX DMA ring architecture documented
- 802.11 association sequence documented (AUTH, ASSOC, 4-way handshake)
- API: init, scan, connect, disconnect, get_connection, power_save, suspend, resume
- WEP rejection (insecure)
- Compile-verified: PASS

#### NEW: ALC285 codec init added to audio.c
- Full widget topology documented (NID 0x01-0x21)
- HP EliteBook pin configuration defaults (0x14=speaker, 0x21=headphone, 0x19=headset mic, 0x12=DMIC)
- HDA verb construction helper (hda_verb, hda_send_verb)
- 9-step init sequence: AFG power, speaker pin, headphone pin, headset mic, DMIC, routing, volume, EAPD, HP quirk
- Processing coefficient 0x10 for HP speaker boost
- Compile-verified: PASS

#### NEW: Suspend/Resume (S14) added to elitebook_x360.c
- hp_elitebook_suspend(): saves kbd backlight, fan speed, smart charge, camera state
- hp_elitebook_resume(): restores all EC state, re-reads battery/thermal, re-probes USB
- hp_elitebook_pm_state(): returns current PM state (ACTIVE/STANDBY/SUSPEND/HIBERNATE)
- Privacy policy re-applied on resume (IR camera disabled)
- Compile-verified: PASS

### Still Open (not in scope for this session)
- P1-02: Thunderbolt MMIO needs HHDM_BASE offset
- P1-03: hp_elitebook_x360_init() not called from kmain()
- P1-05: LAPIC calibration busy-loop ~40x wrong

**Why:** Multiple audit findings (P1-01, P1-04 and peripheral driver gaps) identified missing or incorrect drivers for the HP EliteBook x360 1030 G4. All addressable gaps have been closed.

**How to apply:** All new driver files compile freestanding. USB-dependent peripherals (fingerprint, smart card, NFC) will light up when xHCI/EHCI driver is implemented in Sprint 40+. WiFi will activate when PCI BAR mapping and DMA rings are wired. ALC285 codec verbs will execute when HDA CORB/RIRB DMA is complete (Sprint 36+).
