## Architecture

- PCI discovery: xpci_find_device(8086, 9D2F) or xpci_find_class(0x0C, 0x03) with prog_if=0x30
- BAR0 mapping: HHDM (HHDM_BASE + bar0_phys), same pattern as NVMe driver
- DMA buffers: pmm_alloc_pages(0, PMM_ZONE_DMA32) for below-4GB physical addresses
- Rings: Command ring (256 TRBs), Event ring (256 TRBs), per-slot transfer rings (256 TRBs)
- DCBAAP: 4KB page with uint64_t pointers per slot
- Polling: no MSI/MSI-X, event ring polling for command/transfer completion
- Device enumeration: port reset, Enable Slot, Address Device, GET_DESCRIPTOR
- Control transfers: SETUP+DATA+STATUS TRBs with DMA bounce buffers
- Bulk transfers: Normal TRBs, 4KB max per transfer (multi-page deferred)
- Boot order: Step 10b2 in kmain (after XPCI, before XLAPIC)
