## XKABI Rights Architecture (Verified)
- xkabi_rights.h: SINGLE SOURCE OF TRUTH, 4 consumers aligned
- Software rights: bits 0-11 (GPU_MAP=1, MEM_MAP=3, CONSOLE_OUT=4, FIRMWARE_VAR=5)
- Hardware rights: bits 12-22 (STORAGE=15-16, NET=17-18, GPU_EXEC=19, ATTEST=21)
- DERIVE: bit 23 (sub-capability creation)
- Handle encoding: [63:40]=slot, [39:16]=rights, [15:0]=generation
- Boot cap table: XKABICAT magic, max 128 entries, 4 protocol GUIDs classified
