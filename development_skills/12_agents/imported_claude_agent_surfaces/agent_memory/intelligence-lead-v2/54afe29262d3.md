## Technical Indicator Formula Reference

- Python RSI: `gain.rolling(window).mean()` -- simple rolling mean (feature_engineering.py:119)
- WASM RSI: `avg_gain = (avg_gain * (period-1) + gain) / period` -- Wilder's EMA (statistics.rs:177)
- Python ATR: `true_range.rolling(window).mean()` -- simple rolling mean (feature_engineering.py:203)
- WASM ATR: `(prev * (period-1) + tr) / period` -- Wilder's smoothing (statistics.rs:282)
- Python ADX (standalone): Uses Wilder's smoothing correctly (feature_engineering.py:828-962)
