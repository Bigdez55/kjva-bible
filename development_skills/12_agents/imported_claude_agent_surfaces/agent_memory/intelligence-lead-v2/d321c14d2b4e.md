## Wave 3 ML Audit (2026-03-18, HEAD: 279b28f)

### Numerical Correctness: ALL PRIMITIVES CORRECT (Grade: A)
- xm_expf: degree-5 minimax, max error < 1.5e-6. Coefficients verified = Taylor(2^f at ln2).
- xm_sqrtf: Quake seed 0x1FBD1DF5 + 3x NR. ~6 ULP accuracy.
- xmind_rmsnorm: eps=1e-5, in-place safe (scale computed before writes). Matches LLaMA ref.
- xmind_matmul_q4: Q4_0 nibble order lo/hi matches GGML spec. Dequant = (nibble-8)*scale.
- xmind_softmax: 3-pass numerically stable (max-subtract). Correct.
- xmind_silu: SwiGLU = g * sigmoid(g) * up. Correct.
- xmind_rope: Paired rotation (not interleaved). GQA-aware. Table[pos*half+i]. Correct.
- xmind_forward: Exact LLaMA 2/3 architecture. Pre-norm, residual, SwiGLU, weight-tied logits.
- Attention: GQA mapping h*n_kv/n_heads. Fast inv-sqrt 0x5F3759DF + 1 NR. Causal via KV cache bounds.
- Sampler: xorshift64 (period 2^64-1), IEEE 754 float construction, approx top-p (threshold, not sort).
- Aggregate audit R2=0.95 SIGNAL.

### New Findings
- D-XMIND-P2-01: state.attn alloc n_heads*ctx_len but used as n_heads*head_dim. 240KB waste.
- D-XMIND-P2-02: XMIND_VOCAB_SIZE=32000 is LLaMA-specific. Runtime path uses cfg.vocab_size correctly.
- D-XMIND-P3-01: Top-p is threshold-approximation (conservative), not sort-based nucleus. Intentional for freestanding.

### Model Target Recommendation: Qwen3-0.6B
- RAM: 444 MB vs 2,367 MB (Llama 3.2 3B). Enables 8GB SKU.
- Scalar tok/s: 2.1-3.6 (Qwen 0.6B) vs 0.4-0.6 (Llama 3B).
- AVX2 tok/s: 17-44 (Qwen 0.6B) vs 3-8 (Llama 3B).
- CAVEAT: Qwen3 vocab=151,936 vs XMIND_VOCAB_SIZE=32000. Runtime cfg.vocab_size handles this.
- Minimum viable: 0.6B for language tasks + 5M CNN/MLP for telemetry (dual-tier "immune system").

### SKU Correction
- i5-8265U is Whiskey Lake = AVX2 ONLY. Previous "Ice Lake (AVX-512)" was wrong for this SKU.
- No AVX-512 on HP EliteBook x360 1030 G4 with i5-8265U.

### Performance Gap
- XMIND Q4_0 scalar vs llama.cpp Q4_K_M AVX2: 16-25x gap on same hardware.
- Primary lever: AVX2 SIMD in xmind_matmul_q4. 8-16x gain achievable.
