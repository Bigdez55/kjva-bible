## Observability Infrastructure Summary (Sprint 40 update)

### WIRED (Sprint 40)

- **XSTORE metrics**: xstore_stats_t now includes btree_depth (root page level + 1)
- **XNET stack metrics**: xnet_stack_stats_t aggregate endpoint in netif.c
  - L2: packets_tx/rx, bytes_tx/rx (summed across all interfaces)
  - TCP: tcp_connections_active (live scan), tcp_retransmits, tcp_resets, tcp_timeouts
  - DNS: dns_queries, dns_cache_hits (atomic counters in dns.c)
  - IPv6: ipv6_forwarded, ipv6_dropped (accessors in ipv6.c)
- **Kernel metrics**: xenos_kernel_metrics_t in main.c
  - g_syscall_total, g_syscall_counts[32], g_page_fault_count, g_ctx_switch_count
  - Incremented in xenos_syscall_dispatch(), xenos_isr_dispatch(), xenos_sched_yield()
- **Health endpoint**: genos_health_t / genos_get_health() in main.c
  - uptime_ns, cpu_count, mem_total, mem_used, active_processes, syscall_count
- **Boot timing (GENSD)**: xchain_result_t.stage_duration_ns[10] + total_boot_ns
  - All 10 boot stages timed with pal_time_now_ns()
  - Per-stage timing printed to serial and journaled
- **Boot telemetry (kernel)**: xenos_boot_telemetry_t in main.c (Sprint 46)
  - _rdtsc() inline for cycle-accurate timing, 16 step slots + total
  - Pre-PAL steps use _serial_step_report(); post-PAL use pal_console_printf
  - Accessor: xenos_get_boot_telemetry() for external consumers
  - TSC MHz derived from bi->tsc_hz for cycle-to-us conversion

### PRE-EXISTING

- GENSD journal: 4MB BSS ring, binary packed, fnv1a hash, levels 0-4, RAM-only

- XNET netif: atomic rx/tx counters per interface

## Conventions

- All kernel code uses pal_console_printf (unstructured text to serial)
- XSEC audit events use typed enums, not strings
- GENSD journal uses fnv1a32 hash of service name (not full string)
- PAL API: PAL_SPINLOCK_INIT, pal_spin_lock/unlock, pal_console_printf
- pal_time_now_ns() is the monotonic clock source
- pal_atomic_add/load/store for lock-free counters
