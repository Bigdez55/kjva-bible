### XFRAME C Layer Patterns (Critical)

- **`XTOKEN_COLOR_PRIMARY` does NOT exist** — use `XTOKEN_COLOR_ACCENT` for all interactive/brand color references.
- **`xframe_node_get_child()` is NOT in public API** — store parallel `xframe_node_t *` pointer arrays at init time to enable post-init node text updates.
- **`xframe_node_set_position()` and `xframe_node_set_z_index()` are NOT in public API** — compositor manages surface position; add comment "Production: call compositor surface reposition API".
- **`xframe_node_set_flex_wrap()` is NOT in public API** — use `xframe_node_set_flex_grow(node, 1.0f)` as layout hint instead.
- **`XFRAME_JUSTIFY_SPACE_BETWEEN`/`XFRAME_JUSTIFY_CENTER` do NOT exist** — `xframe_node_set_justify_content()` takes `xframe_align_t` (XFRAME_ALIGN_START/CENTER/END/STRETCH).
- **Forward declarations required** when a public function calls a sibling public function defined later in the same TU (e.g., `_next()` calling `_hide()`).
- **`__attribute__((unused))`** on helper functions that exist for completeness but are not called in the minimal freestanding build — avoids -Wunused-function under -Werror.
- **orange_store.c is single-TU** — `#include`'d directly by xshell.c; must NOT be in XSHELL_SRCS (double-definition link error).
