# Chart Migration — Line Charts Only (2026-02-20)

## Migration Summary
Replaced ALL candlestick charts and bar charts with simple, clean line charts across the entire application.

## Components Replaced
1. **CandlestickChart** → **LineChart** (Trade tab in InvestPage)
2. **PortfolioChart** → **LineChart** (Dashboard + InvestPage Portfolio tab)

## New LineChart Component
- **Location:** `frontend/src/components/elson-v2/charts/LineChart.tsx`
- **Features:**
  - Simple line graph with smooth curves
  - Gradient area fill (optional, controlled by `showGradient` prop)
  - End-point dots (optional, controlled by `showDots` prop)
  - Color coding: green for gains, red for losses
  - Grid lines for reference
  - Time labels based on timeRange
  - Price labels (min/max)
  - Loading and error states
  - WASM-accelerated scaling with JS fallback
  - Responsive design with `viewBox` and `preserveAspectRatio`

## Props Interface
```typescript
interface LineChartProps {
  data: LineChartDataPoint[];
  isLoading?: boolean;
  isError?: boolean;
  timeRange?: string;
  showGradient?: boolean;
  showDots?: boolean;
  height?: number;
  containerHeight?: number;
}

interface LineChartDataPoint {
  value: number;
  timestamp?: string;
}
```

## Implementation Patterns

### Historical Stock Data (Trade Tab)
```typescript
// Fetch historical data via RTK Query
const { data: historicalData, isLoading, isError } = useGetHistoricalDataQuery(
  { symbol: sel.symbol, period: chartPeriodMap[chartTimeframe] },
  { skip: !sel.symbol || sel.symbol === '—' }
);

// Transform to LineChart format
const chartData: LineChartDataPoint[] = useMemo(() => {
  if (!historicalData?.data) return [];
  return historicalData.data.map(point => ({
    value: point.close,
    timestamp: point.timestamp
  }));
}, [historicalData]);

// Render
<LineChart
  data={chartData}
  isLoading={isLoading}
  isError={isError}
  timeRange={chartTimeframe}
  showGradient={true}
  showDots={true}
/>
```

### Portfolio Value Data (Dashboard + Portfolio Tab)
```typescript
// Generate synthetic portfolio data
const portfolioChartData: LineChartDataPoint[] = useMemo(() => {
  const counts: Record<string, number> = {
    '1D': 24, '1W': 7, '1M': 30, '3M': 12, '1Y': 12, ALL: 24,
  };
  const count = counts[timeRange] || 12;
  const startValue = total * 0.83; // Assume 17% gain
  const endValue = total;

  const points: LineChartDataPoint[] = [];
  for (let i = 0; i < count; i++) {
    const t = i / Math.max(count - 1, 1);
    const value = startValue + (endValue - startValue) * t;
    const noise = (Math.random() - 0.5) * (endValue - startValue) * 0.02;
    points.push({ value: value + noise });
  }
  return points;
}, [timeRange, total]);

// Render
<LineChart
  data={portfolioChartData}
  isLoading={isLoading}
  isError={!!error}
  timeRange={timeRange}
  showGradient={true}
  showDots={true}
  height={120}
  containerHeight={140}
/>
```
