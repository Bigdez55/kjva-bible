## Architecture Pattern — Full-Screen Conditional Overlay
InvestPage uses a `profileSymbol: string | null` state.
When non-null, an early return renders `<StockProfilePage>` INSTEAD of the normal invest tab content.
This avoids React Router and keeps the single-page tab architecture intact.

```tsx
// Early return BEFORE the main JSX return, AFTER all hooks:
if (profileSymbol) {
  return (
    <StockProfilePage
      symbol={profileSymbol}
      onBack={() => setProfileSymbol(null)}
      onTrade={(sym) => {
        setProfileSymbol(null);
        const matchedStock = stocks.find((s) => s.symbol === sym);
        if (matchedStock) selectStock(sym);
        setSub('trade');
      }}
    />
  );
}
```

The Discover tab stock grid cards trigger entry:
```tsx
onClick={() => setProfileSymbol(s.symbol)}
```

## Data Queries Used
- `useGetStockProfileQuery(symbol)` — company info, stats, news
- `useGetHistoricalDataQuery({ symbol, period })` — chart data
- Period mapping: `{ '1D': '1d', '1W': '5d', '1M': '1mo', '3M': '3mo', '1Y': '1y' }`

## Layout Structure (top to bottom)
1. Back button row — chevron left + symbol name + logo/avatar
2. Price header Card — large price, change %, Buy/Sell action buttons
3. Chart Card — timeframe pill selector + LineChart component
4. Daily stats strip — Open / Prev Close / Volume in an Inner row
5. About Card — collapsible description (280 char limit) + StatRow metadata
6. Key Stats Card — 2-column KeyStatGrid with Inner items
7. News Card — up to 5 news items as `<a target="_blank">` wrapped Inner rows

## Sub-components (defined in same file)
- `StatRow` — label/value flex row with `C.glassBorder` bottom border
- `KeyStatGrid` — 2-col grid of Inner boxes with optional color override
- `ProfilePageSkeleton` — full skeleton layout for loading state

## Key UX Decisions
- Buy button: gold gradient, C.bg text — primary action
- Sell button: transparent with `C.red` border — secondary/destructive
- Both buttons call `onTrade(symbol)` — the trade dialog in InvestPage handles buy/sell distinction
- Description truncated at 280 chars with "Read more" toggle (no animation needed)
- News items: entire Inner is wrapped in `<a>` — full tap target, no onclick JS
- Logo fallback: gold gradient div with `symbol.slice(0, 2)` text — never shows broken img
- Error state: full centered error with icon + "Go Back" button
- Skeleton `@keyframes profileSpin` injected via `<style>` tag in component render

## Number Formatting Utilities (defined in file)
- `formatLargeNumber(n)` → `$1.23T / $456.78B / $12.34M`
- `formatVolume(n)` → `1.23B / 456.78M / 12.3K`

## TypeScript: Zero Errors
- All optional fields on `StockProfile` handled with `?? '—'` fallback
- `profile?.field != null` guards before `.toFixed()` calls
- `useMemo` for chartData transformation — handles empty `historicalData?.data`
- No `any` types used anywhere in the file
