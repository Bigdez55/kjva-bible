## Auth Chain Pattern (Confirmed 2026-02-18, updated 2026-03-08)

- JWT stored in localStorage across 20+ frontend files — XSS risk (P1)
- httpOnly cookies ARE set on login but NOT relied on by frontend service files
- `/auth/logout` does NOT clear httpOnly cookie — only client-side localStorage.removeItem
- 2FA implemented (pyotp TOTP, /security/2fa/verify) but NOT enforced at login — opt-in only
- Argon2 is primary password hash (CryptContext schemes=["argon2","bcrypt"])
- Token revocation via JTI in Redis — silently skipped when Redis is down

- FernetEncryptor correctly uses ENCRYPTION_SALT (separate from JWT key)
- No PII in EFT portfolio_context — CONFIRMED
