### Architecture Decisions
- Panic handler uses ONLY static buffers (no heap/locks in panic path)
- NMI broadcast via LAPIC ICR shorthand all-except-self (0xC0400)
- Watchdog timeout: 30s, TSC-based, checked from LAPIC timer at 100 Hz
- OOM recovery ladder: reclaim -> kill -> panic (cooperative sched cannot force-kill yet)
- Stack canaries: dual 8-byte pattern (DEADC0DE5AFEC0DE / BADCAFE0FEEDFACE)
- LAPIC timer handler now composite: watchdog_pet + canary_check(1Hz) + xnet_tick
