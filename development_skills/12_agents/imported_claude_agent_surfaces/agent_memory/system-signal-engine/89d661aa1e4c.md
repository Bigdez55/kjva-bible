## NEW Regressions Introduced by 045f4e6

### REGRESSION-1: ThermalGuardian constructor crashes on non-Linux (P1-MEDIUM)
- `STATE_PATH.parent.mkdir(parents=True, exist_ok=True)` added to `__init__`
- `/run/genos/` is read-only on macOS -> OSError: [Errno 30]
- Causes 30+ thermal/fabric/merkle test failures (cascading)
- Pre-fix code did NOT have eager mkdir in constructor
- Fix: defer mkdir to `_write_state()` or guard with try/except

### REGRESSION-2: meshd Ed25519 private key not zeroized on stop (P2-SECURITY)
- `genos_meshd_stop()` does not call `memzero_explicit(meshd.my_privkey, 32)`
- Private key persists in kernel memory after module unload
- Pre-existing issue but not flagged in previous audit

### REGRESSION-3: JIT cache eviction leaks LLVM native code (P2-MEMORY)
- `xisc_jit_get_fn()` overwrites cache entry without freeing old compiled code
- No `LLVMRemoveModule` or equivalent on eviction path
- Under sustained workload switching, unbounded memory leak
