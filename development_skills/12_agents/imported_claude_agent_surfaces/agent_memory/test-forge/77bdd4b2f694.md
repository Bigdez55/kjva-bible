### CRITICAL FUZZ CI FLAW
Every fuzz invocation ends with `|| true`:
```

```
Any fuzz crash does NOT fail the job. The CI captures artifacts but ALWAYS passes.
Per the comment: "capture the corpus/crash artifacts instead of failing silently."
Net effect: fuzz job is a compile+run-30s check, never a regression gate.
