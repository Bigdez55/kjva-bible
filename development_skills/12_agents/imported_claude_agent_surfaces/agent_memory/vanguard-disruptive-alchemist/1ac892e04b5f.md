### Key Forensic Evidence
- Update service: 119-line stub, in-memory dict, no rollback
- Undo tool: "not yet implemented, Phase 6.2"
- packages.list: missing orca, at-spi2-core, upower, wlr-randr
- 10 separate Electron binaries = 10 copies of Chromium
- No tiling/snap/workspace code anywhere in compositor
- Settings menu item exists but links to nothing
- FirstRunWizard: free-text inputs, no privacy consent
- ARIA markup: genuinely excellent (role/aria-label/aria-pressed everywhere)
- PII redaction: genuinely novel (SSN, CC, email, phone patterns)
- Local AI: genuine differentiator (Ollama Llama 3.2 3B on-device)

### Root Cause
Sequencing failure: deep infrastructure (kernel, XKABI, vDRAM, mesh, entropy)
invested before user-visible surface (window mgmt, settings, a11y, updates).
Caterpillar with butterfly DNA -- metamorphosis has not occurred.

### Switching Cost/Benefit: 9:6 (costs dominate)
No user would switch today. Cost: no app compat, no window mgmt, no settings,
no rollback, no screen reader, 2.6-3.5GB base RAM.
Benefit: local AI, free, simple setup, PII redaction, thermal monitoring.
