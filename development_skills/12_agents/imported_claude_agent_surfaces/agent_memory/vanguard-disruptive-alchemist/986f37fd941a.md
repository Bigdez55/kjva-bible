- Llama 3.2 3B at scalar: ~20 seconds per token (UNUSABLE)
- Fix: SSE4.2/AVX2 intrinsics in matmul_q4/dot/softmax (~200 LOC, 33-133x)

- Llama 3.2 3B = 2.1 GB RAM (13% of 16GB) -- too heavy for OS assistant

- Architecture is model-agnostic (config params only) -- switch is ~50 LOC

### AES-NI TODO (P0 SEC, 13 Sprints Old)

## XJIT Honest Assessment
- 22 integer opcodes, ZERO float/SIMD -- serves XEMU, NOT XMIND
- One-pass compilation: correct for short translation blocks
- Linear-scan regalloc: O(n^2) but bounded at 256 vregs
- NOT a general-purpose JIT. NOT an LLVM replacement. Reframe narrative.

## Top 5 Sacred Cows (Still Standing)
