## 3 SOTA Gaps

### GAP 1 (P2): Tool files use per-call httpx (no connection pooling)
- browser_tool.py, explain_tool.py, file_tool.py use httpx.post()/get() bare
- Adds ~50-100ms TCP handshake per call vs singleton pattern used in agent.py
- Fix: add module-level _client = httpx.Client(timeout=10.0)

### GAP 2 (P3): Dead requests==2.32.3 dependency
- Zero imports of requests anywhere in codebase (confirmed via full grep)
- Adds ~2.5MB bloat + transitive CVE surface to all Docker images
- Fix: delete line 11 from platform/services/requirements.txt

### GAP 3 (P2): No SSE streaming for AI responses
- Ollama uses "stream": False -- entire response buffered before return
- 512 tokens at ~30 tok/s = ~17s perceived latency before first output
- SSE streaming would deliver first-token in ~200ms (85x improvement)
- Highest-leverage Time-to-Value optimization available
