## Competitive Intelligence

### AES constant-time
- No production library uses 256-entry linear scan
- Industry standard: AES-NI hardware (preferred) or bitsliced software (BearSSL)
- BearSSL ct_aes is the reference for provably-CT software AES

### TLS 1.3
- BearSSL is gold standard for small-footprint (~15K LoC, full X.509, ~6KB heap)
- s2n-tls (AWS) is gold standard for production correctness
- Our 1.1K LoC TLS cannot interop with real servers due to SHA mismatch + no cert validation

### TCP
- lwIP is the direct competitor (full-featured embedded TCP/IP)
- Our TCP is comparable to uIP (minimal) not lwIP (production)
- CUBIC is standard since Linux 2.6.19 (2006), RFC 9438 (2023)

### Package Security
- 9-gate refusal machine: confirmed Blue Ocean, no equivalent anywhere
- Gaps: no sigstore-style transparency log, no TUF threshold signing
- Content-addressed store (Nix-style) would be high-value Sprint 5+ addition
