- Python suite (xisc/conformance/): tests XKABI object model, handle lifecycle, rights monotonicity,

  (conftest.py + test_objects.py + test_memory.py + backends/linux_shim.py)
- New C suite (xkabi_conformance.c): tests capability handle table, derivation, invocation,
  revocation, epoch bulk revocation, RDMA capability barriers, performance (26 tests)
- VERDICT: COMPLEMENTARY, not conflicting. C tests validate the raw capability security model at the
  bit-flag level. Python tests validate the higher-level XKABI object model (VMO, channels, events,
  rights, signals). They cover different abstraction layers.
