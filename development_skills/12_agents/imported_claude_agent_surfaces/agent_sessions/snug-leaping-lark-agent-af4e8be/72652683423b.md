## 3. The Audit (Feedback Loop)

### CRITICAL Issues

#### C-1: Excessive `any` Type Usage (83 Files)
- **What:** 83 TypeScript files contain explicit `any` type annotations or `as any` casts
- **Where:** Key offenders include:
  - `frontend/src/store/slices/authSlice.ts:48` - `catch (error: any)`
  - `frontend/src/store/slices/marketDataSlice.ts:28,40,56,68` - Multiple `error: any`
  - `frontend/src/hooks/useMarketWebSocket.ts:45-46` - `(update as any).high_24h`
  - `frontend/src/services/auth.ts:17,26` - `user: any` in response types
  - `frontend/src/store/middleware/websocketMiddleware.ts:41,44,52,59` - Multiple `any` in handlers
- **Why:** Loss of type safety in a fintech application handling financial data. Runtime errors may go undetected until production.
- **How:** Replace with proper typed error handling:
```typescript
// Before (authSlice.ts:48)
} catch (error: any) {
  return rejectWithValue(error.response?.data?.detail || 'Login failed');
}

// After
} catch (error) {
  if (axios.isAxiosError(error)) {
    return rejectWithValue(error.response?.data?.detail || 'Login failed');
  }
  return rejectWithValue('Login failed');
}
```

### HIGH Issues

#### H-1: parseFloat for Financial Calculations (49 Instances)
- **What:** `parseFloat()` used for monetary value parsing throughout the codebase
- **Where:**
  - `frontend/src/components/trading/DollarBasedInvestment.tsx:64-65,80-81,101,105,117,128`
  - `frontend/src/components/trading/OrderForm.tsx:110-112,130-131`
  - `frontend/src/components/trading/QuickInvestButtons.tsx:105,128,273,297`
  - `frontend/src/components/portfolio/PortfolioBuilder.tsx:232,809,812,840`
  - `frontend/src/pages/TransfersPage.tsx:197`
- **Why:** Floating-point arithmetic can cause precision errors in financial calculations (e.g., $0.10 + $0.20 !== $0.30 in IEEE 754).
- **How:** Use integer cents or a decimal library:
```typescript
// Before
const shares = parseFloat(amount) / data.price;

// After - use cents internally
const amountCents = Math.round(parseFloat(amount) * 100);
const shares = amountCents / (data.price * 100);
```
